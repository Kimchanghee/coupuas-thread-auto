# Changelog

## v3.0.0 — Claude Dark UI 환골탈태 + 어그로 생성 v2 (2026-05-04)

### 디자인 시스템 전면 교체
- "Stitch Blue" 다크 테마 → **"Claude Dark"** (warm charcoal + coral) 전면 교체
- 모든 블루 RGB(`13,89,242 / 59,123,255`)를 코랄 RGB(`217,119,87 / 232,145,117`)로 일괄 치환
- `theme.py`의 Colors / Gradients / 컴포넌트 헬퍼 / 글로벌 스타일시트 재구성
- 세이지 그린(success) / 웜 앰버(warning) / 웜 레드(error) / 웜 블루(info) — 전 시맨틱 톤도 차분하게 통일

### 메인 윈도우 레이아웃 재설계
- 윈도우 1280×800 → **1360×960**, 헤더 68 → 84, 사이드바 280 → 320
- 사이드바 정보를 **카드 3개** (진행 상황 / 통계 / 작업 로그)로 그룹화
- 통계 카드: `성공: 0  실패: 0  전체: 0` 한 줄 → **18pt 큰 숫자 가로 3분할** (그린/레드/코랄 색상 코딩)
- 페이지 헤더 칩 36 → 44 + 코랄 그라디언트 + 외곽선, 타이틀 15pt → 20pt
- 모든 페이지 카드 너비 944 → **984** (새 컨텐츠 폭 기준 우측 정렬)
- 사이드바 ↔ 상태바 40px 침범 버그 수정

### 튜토리얼 오버레이 환골탈태
- 도트 11개 → **카드 상단 풀폭 진행률 바** (코랄 그라디언트, 단계별 채움 시각화)
- "1 / 11 단계" → **"STEP 1 OF 11"** 코랄 영문 캡션
- 카드 폭 340 → **540**, 본문 line-height HTML `160%` 적용 + `desc_label.heightForWidth()`로 잘림 원천 차단
- 모든 컨트롤(이전/건너뛰기/다음/체크박스)을 **카드 한 덩어리** 안에 통합 (떠다니는 위젯 제거)

### 다이얼로그 / 팝업 일관화
- `update_dialog.py` — 코랄 그라디언트 액션 + 고스트 보조 위계
- `ui_messages.py` — OS 기본 파란/노란 아이콘 제거(`Icon.NoIcon`), `예`(코랄) ↔ `아니오`(고스트)로 시각적 위계
- `settings_dialog.py` 헤더 그라디언트도 새 톤으로

### 픽셀 단위 잘림 수정 (자동 검수 도구로 잡음)
- "업데이트 확인" 버튼 — 한국어 6자 잘림 → ghost 버튼 padding 22 → 14 (좌우 56 → 28) + 너비 확장
- "튜토리얼 재실행" / "문의하기" — 코랄 페이지 헤더 액션 버튼들 텍스트 좌우 잘림 해결
- 헤더 알약 버튼 (`로그아웃 / Tutorial / 업데이트`) — sizeHint + 16 → +30 마진
- Threads 계정 카드 — `● 연결 안됨` 라벨이 로그인 버튼과 4px 겹침 → 카드 184 + 버튼 y=140
- API 카드 — `마스터 API 키` 라벨이 `Google AI Studio에서 발급` 부설명 74px 덮음 → 라벨 너비 100
- 텔레그램 카드 — 채팅 ID 입력란이 카드 boundary 밖 2px 삐져나감 → 카드 192
- 페이지 2 ScrollArea content가 viewport보다 8px 큼 → `setFixedWidth(CONTENT_W - 16)`
- 사이드바 진행 상황 카드 도트(●) 글자 16 → 24 (5px 부족 해결)
- 업로드 설정 spinbox — 텍스트와 화살표 분리감 해결 (subcontrol-position + padding 재구성)
